## 🌟 Moment of Memory 소개

- OTT 서비스가 대중화 되면서 수 많은 사람들이 하루에 영화, 드라마 등 많이 챙겨 봅니다.
- 누군가에겐 그저 보고 흘러가는 순간 일 수 있지만, 또 다른 누군가는 그 때의 감정과 순간을<br> 기록 하고 싶을 수 있습니다.
- **Moment of Memory** 는 그 순간의 기억과 감정을 기록할 수 있게 도와주는 웹 앱 입니다.
